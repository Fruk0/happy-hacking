## El engagement

A lo largo del curso trabajás como consultor recién contratado para realizar un pentest completo sobre **Nexus Financial S.A.** — una fintech ficticia que usamos como hilo narrativo.

## Uso ético

Todo el material de este curso está orientado a pentest **con autorización explícita**.

Aplicar estas técnicas contra sistemas sin permiso es ilegal en Argentina (Ley 26.388) y en la mayoría de las jurisdicciones. El objetivo del curso es prepararte para trabajar de forma profesional en seguridad ofensiva.
